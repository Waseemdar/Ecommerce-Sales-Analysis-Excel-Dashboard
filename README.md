# E-Commerce Sales Analysis Dashboard (Excel)

## Project Overview
This project is a comprehensive **Excel-based e-commerce analytics dashboard** built to analyze large-scale transactional data and convert it into clear, decision-ready insights.  
The dashboard focuses on sales performance, profitability, customer behavior, and geographic distribution using advanced Excel features.

## Dataset
- ~110,000+ transaction records
- 21 business attributes including:
  - Orders, Customers, Sales, Profit
  - Product Categories
  - Markets & Countries
  - Customer Segments
  - Order Status & Shipping Modes

## Key KPIs
- Total Orders: ~41K  
- Unique Customers: ~17.5K  
- Total Sales: ~26.4M  
- Total Profit: ~2.49M  
- Average Order Value: ~645  

## Key Insights
- Sports and outdoor categories (Cardio Equipment, Fishing, Camping & Hiking) are the highest revenue contributors.
- Europe is the top-performing market, followed by LATAM and Pacific Asia.
- Consumer segment drives the majority of revenue, outperforming Corporate and Home Office segments.
- Revenue concentration suggests opportunities for category-level optimization and regional growth strategies.

## Dashboard Components
- **KPI Summary Sheet** – High-level business metrics
- **Trend Analysis** – Time-based sales and order movement
- **Market Distribution** – Region and country-level performance
- **Customer Segmentation** – Revenue by customer type
- **Top 5 Analysis** – Best-performing products and regions

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Advanced Excel formulas
- Data cleaning and transformation
- Business KPI design
- Dashboard storytelling

## Why This Project Matters
This dashboard demonstrates how **Excel alone** can be used to:
- Handle large datasets
- Perform structured business analysis
- Deliver executive-level insights
- Build dashboards aligned with real-world decision-making
